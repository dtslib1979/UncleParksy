왜 한 채널(코드팩토리) 집중이 유리한가?
  • 커리큘럼 연속성 유지로 누적 학습이 된다.
  • 포트폴리오 직결 산출물 확보가 쉽다.
  • AI 보조(ChatGPT)로 난이도·속도 보정이 가능하다.
태블릿만으로 가능한 최대치는?
  • 브라우저 기반 실습(Colab, Codespaces) 접속, 기록·요약·리뷰.
  • 경량 추론(TFLite/CoreML) 테스트 수준. 학습(Training)은 클라우드/PC 필요.
최소 커리큘럼만으로도 충분한가?
  • Python 기초 + API + Data I/O + 프로젝트 반복이면 실용 목적에는 충분.
  • 대규모 시스템 설계/CS 이론은 필요시 별도 보강(CS50 등) 권장.
