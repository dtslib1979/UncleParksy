프롬프트 템플릿 10 (질의응답형 아코디언)

말로 코딩 템플릿 — details/summary

1) 환경 세팅 — Initialize → Configure → Install → Set up 
새 Python 프로젝트 Initialize하고 가상환경 Configure해. 필수 라이브러리 Install하고 SQLite DB Set up해.
2) 데이터 처리 — Design → Migrate → Insert → Query 
User 테이블 Schema Design 후 CSV 데이터 Migrate. Seed 데이터 Insert하고 이름 기준으로 Query.
3) 백엔드 API — Implement → Handle Request → Process → Respond 
로그인 API Implement. POST 요청 Handle → 비밀번호 Validate Process → JWT Respond.
4) 프론트 — Render → Fetch → Bind → Handle Event 
React로 로그인 UI Render. API Fetch → 상태 Bind → 로그인 버튼 Handle Event.
5) 로직 — Validate → Compute → Sort → Optimize 
입력 Validate → 평균 Compute → 점수 Sort → 상위 10 Optimize.
6) 테스트 — Test → Mock → Assert → Review 
회원가입 API Test. 가짜 유저 Mock → 상태코드 200 Assert → 결과 Review.
7) 디버깅 — Debug → Trace → Profile → Fix 
서버 Debug 모드. 호출 Trace → CPU Profile → 메모리 누수 Fix.
8) 배포 — Build → Package → Deploy → Monitor 
프로젝트 Build → Docker Package → AWS Deploy → CloudWatch Monitor.
9) 보안 — Scan → Patch → Harden → Audit 
취약점 Scan → SQLi Patch → 서버 Harden → 접근 로그 Audit.
10) 협업 — Branch → Commit → Merge → Resolve Conflict 
새 기능 Branch → 코드 Commit → 메인으로 Merge → 충돌 Resolve.
