🚀 UncleParksy 작동 원리

파일 업로드만 하면 자동으로 웹페이지 발행!

1
📝 HTML 파일 작성
로컬 또는 GitHub 웹 에디터에서 HTML 파일을 작성합니다.
템플릿을 사용하면 더 쉽습니다!
↓
2
📤 GitHub에 업로드
category/ 폴더 내 원하는 카테고리에 파일을 업로드합니다.
Git 명령어 또는 웹 인터페이스 사용 가능!
↓
3
🤖 자동 처리 시작
GitHub Actions가 자동으로 실행됩니다:
  • sitemap.xml 업데이트
  • 카테고리 인덱스 정렬
  • 변경사항 자동 커밋
↓
4
🌐 GitHub Pages 배포
GitHub Pages가 자동으로 사이트를 재배포합니다.
약 30초-2분 소요 ⏱️
↓
5
✅ 발행 완료!
웹사이트에서 새 페이지를 확인할 수 있습니다:
https://parksy.kr/category/카테고리명/파일명.html
