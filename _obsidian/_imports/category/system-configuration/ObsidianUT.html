Production Workflow

Four stages from ideation to deployment.

1

Sketch

Draft vectors in Excalidraw; isolate text vs visuals. Export transparent PNG/SVG to your CDN.

2

Map Canvas

Build a "treasure map" with zoom levels (overview 25%, chapter 60%, detail 120%). Link nodes to notes/media.

3

Integrate CDN

Reference external assets only. Keep the vault lightweight and portable.

4

Publish

Deploy via Obsidian Publish or a static host. Validate mobile UX and accessibility.

Essential Setup

Hardware, plugins, accounts ‚Äî with practical steps.

üì±

Hardware

  • Tablet + stylus (Galaxy Tab + S-Pen recommended, iPad + Pencil works too).
  • Built-in screen recorder (1440p/60fps suggested) for tutorials.
  • Enable palm rejection; reduce gesture conflicts.
üîå

Plugins (Obsidian)

  1. Core: Canvas (enable in Settings ‚Üí Core plugins)
  2. Community: Excalidraw, Media Extended, Templater
  3. Optional: Advanced Tables, Style Settings + Minimal Theme, Graph Analysis

Keep plugin count minimal to avoid mobile slowdown.

‚òÅÔ∏è

CDN & Domains

  • GitHub Pages / Netlify (static), Cloudinary (image/video transforms)
  • Name assets with versions: cover_v4.png
  • Test public access in an incognito window.

Obsidian Publish (optional) ‚Äî Quick Start

  1. Create a /_publish folder; put public pages inside.
  2. Settings ‚Üí Publish ‚Üí Select Vault folder(s) ‚Üí Publish.
  3. (Optional) Connect a custom domain via CNAME per provider docs.

Prefer zero-cost? See Alternatives.

Troubleshooting & Performance

Most common issues, quick fixes, and safeguards.

üîó

CDN links not visible after publish

  • Open the URL in an incognito window (check 403/404/CORS).
  • Ensure the asset is public; avoid signed/expired URLs.
  • Cache-bust with ?v=4 or increment file versions.
üìâ

Canvas feels slow on mobile

  • Split one giant board into overview/detail boards.
  • Reduce embedded previews; link out where possible.
  • Disable unused plugins; restart app to clear memory.
üñäÔ∏è

Excalidraw stroke issues

  • Increase zoom while drawing; export at 2x.
  • Fix pen thickness presets (thin/medium/bold) and one highlight color.
  • Separate text and visuals onto distinct layers.

Accessibility & SEO

  • Add alt text for meaningful images; use captions for complex diagrams.
  • Maintain heading order (h1‚Üíh2‚Üíh3), keyboard focus, and color contrast.
  • Static titles/metadata per page; keep link labels descriptive.

Privacy & Licensing

  • Prefer on-device AI tools for sketch assists when available.
  • Ensure you have rights to all media; track licenses in a note.
  • Avoid embedding sensitive data in public vaults.
